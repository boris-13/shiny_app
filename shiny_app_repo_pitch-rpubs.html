Embeded Code in the Slide

  • The following code calculates mpg prediction on given weight based on minimal linear regression model
  • The output shows the predicted mpg value and the lower and upper limits of the confidence interval
fit <- lm(mpg ~ wt, data = mtcars)

predict(fit, newdata = data.frame(wt = 3.45), interval="confidence")

      fit     lwr    upr

Moto Trend Shiny Application: Dataset

  • For this assignment, we used the mtcars dataset.
  • The data was extracted from the 1974 Motor Trend US magazine, and comprises fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (197374 models).
